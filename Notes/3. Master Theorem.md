# 3. Master Theorem

Let $a, C > 0$ and $b > 0$ be constants and $T : \mathbb{N} \to \mathbb{R}^+$ a function such that for all even $n \in \mathbb{N}$,
$$T(n) \leq aT \left( \frac{n}{2} \right) + Cn^b$$

Then for all $n = 2^k$, $k \in \mathbb{N}$, the following statements hold

1. If $b > \log_2 a$, $T(n) \leq O(n^b)$.
2. If $b = \log_2 a$, $T(n) \leq O(n^{\log_2 a} \cdot \log n)$.
3. If $b < \log_2 a$, $T(n) \leq O(n^{\log_2 a})$.

If the function $T$ is increasing, then the condition $n = 2^k$ can be dropped


## Explanation

If work goes down every layer
$\implies$ work of top layer (lower layers become asymptotically negligible)

If work increases every layer
$\implies$ work of bottom layer (top layers become asymptotically negligible)

If work stays constant
$\implies$ work of all layers (nothing becomes asymptotically negligible)
